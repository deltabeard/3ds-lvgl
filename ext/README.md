# External Dependencies

This folder contains external dependencies that are required for this project to compile.

<dl>
  <dt>MSVC_DEPS.cab</dt>
  <dd>A compressed Windows Cabinet (CAB) file that contains prebuilt libraries required to compile this project with Visual Studio for Win32 and Win64 platforms.</dd>
  <dt>3ds_cia_template.rsf</dt>
  <dd>A template used for generating CIA executables for the 3DS.</dd>
  <dt>logo.bcma.lz</dt>
  <dd>A pre-compiled animation used for the CIA executable on the 3DS.</dd>
</dl>
